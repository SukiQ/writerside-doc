# 安装

## 安装插件
打开 IntelliJ IDEA，选择 `File` -> `Settings` -> `Plugins` -> `Marketplace`，搜索 `Writerside`，安装并重启 IDE


<img src="install-0.png"  thumbnail="false" width="800" alt="install-0"/>

## 创建项目
选择 `File` -> `New` -> `Project` -> `Writerside` 

<img src="install-1.png"  thumbnail="false" width="800" alt="install-1"/>

## 创建主题
选择 `Table of Contents` -> `New` -> `Empty MD Topic`

<img src="install-2.png"  thumbnail="false" width="500" alt="install-2"/>
