# 简介



## 什么是 Writerside？

- **Writerside** 是 JetBrains 基于 IntelliJ 平台打造的全功能文档创作与发布 IDE，旨在统一编写、测试、构建和发布技术文档的流程
- 它可作为独立应用使用（**现已废弃**），也以插件形式集成在 JetBrains IDE 中，如 IntelliJ IDEA

> JetBrains 于 2025 年 3 月发布公告，称已决定 **停用 Writerside 独立 IDE**
{style="note"}

## 核心功能

### 1. “文档即代码”的流水线

采用类似开发流程的方式管理文档，包括 Git 控制、PR 评审，以及内置的自动检查功能，实现真正意义上的“doc as code”

### 2. Markdown 与 XML 双格式混写

支持 Markdown 与 JetBrains 自定义的 XML 标记语言，二者可灵活混用。例如，你可以在 Markdown 中嵌入语义属性，也可将 Markdown 片段实现 XML 转换 

### 3. 内置测试与样式模板

提供超过 100 项内建测试（例如破链、缺少资源、属性校验等），并支持内部风格或拼写规范自定义。同时提供可定制的预设文档样式，无需额外处理布局或 CSS 

### 4. 实时预览 + AI 拼写与语法检查

编辑时提供实时渲染预览，即时反馈错误。还附带 JetBrains 自研的 AI 拼写和语法检查，支持超过 25 种语言
